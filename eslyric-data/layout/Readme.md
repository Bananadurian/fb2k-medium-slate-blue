# Readme

自定义布局和第三方布局。

专辑封面主要使用遮罩Mask实现！

## 文件夹说明

- `design`: 使用AI生成的设计稿件，其中有几个静态网页是用于生成遮罩图片的，如`Vertical Vinyl Pro UI1 1.0-黑胶摩尔纹等遮罩素材生成.html`。
  - `AlphaMaskTool.html`: 生成简单遮罩图的纯本地工具，该工具主要用于设计ESLyric封面。

- `imgs`: 个人定义布局使用的图片素材。
  - `icon`: 图标
  - `mask`: 遮罩图片

- `Style1`: 该布局样式来自仓库[ESLyric-full-screen-imitation-IOS-configuration](https://github.com/gebilaowang66662/ESLyric-full-screen-imitation-IOS-configuration/blob/main/README.zh-CN.md)，*目前该文件夹已删除！！*
